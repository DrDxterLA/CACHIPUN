# 🎮 Juego Cachipún - Usuario vs Consola

Proyecto simple en HTML, Bootstrap y JavaScript que implementa el juego
Cachipún (Piedra, Papel o Tijera).

## 📌 Tecnologías usadas
- HTML5
- Bootstrap 5
- JavaScript Vanilla

## 🎯 Funcionamiento
- El usuario selecciona piedra, papel o tijera.
- La consola elige una opción de forma aleatoria usando Math.random().
- El sistema evalúa el resultado y lo muestra en pantalla.

## 🧠 Características
- Elección de la consola no predecible
- Interfaz simple y responsiva
- Código comentado y fácil de entender

## ▶️ Cómo ejecutar
1. Descargar el proyecto
2. Abrir `index.html` en el navegador
3. Jugar 🎉
